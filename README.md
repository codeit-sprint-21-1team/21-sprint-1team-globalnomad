# 🌏 특별한 체험 탐색과 예약, GlobalNomad

**Global Nomad**는 전 세계의 다채로운 체험을 한눈에 탐색하고, 나만의 소중한 경험을 기록하는 플랫폼입니다.  
지도와 필터로 나만의 체험을 발견하는 **탐색**부터, **스마트한 예약 관리**와 **경험의 기록**까지 **사용자들의 모든 순간을 서포트**하도록 만들었습니다.

## ⚡ 주요 기능

### 체험 활동 (Guest)

- [x] **체험 탐색:** 카테고리 필터링, 가격순/최신순 정렬 및 검색어를 통한 맞춤형 체험 찾기
- [x] **실시간 예약:** 캘린더와 타임슬롯을 활용한 날짜별 예약 신청 및 참여 인원 기반 자동 금액 산정
- [x] **활동 관리:** 마이페이지 내 실시간 예약 상태 트래킹(신청/취소/승인/거절/완료) 및 후기 작성

### 호스트 활동 (Host)

- [x] **체험 관리:** 내 체험 등록, 상세 정보 수정 및 삭제
- [x] **예약 대시보드:** 캘린더 뷰를 통해 등록한 체험의 날짜별 예약 현황 확인 및 수락/거절 관리
- [x] **이미지 최적화:** 메인 배너 및 다중 소개 이미지 업로드 및 관리

### 공통 서비스 및 기술 구현

- [x] **인증 및 보안:** JWT 기반 인증 및 카카오 소셜 로그인 연동
- [x] **유효성 검사:** Zod를 활용한 실시간 폼 검증 및 커스텀 에러 피드백
- [x] **사용자 경험:** 전 페이지 반응형 디자인 적용
      <br><br>

## ✨ 추가 기능

- [x] **비밀번호 강도 체크:** 실시간 보안 강도 검증 및 프로그레스 바 시각화
